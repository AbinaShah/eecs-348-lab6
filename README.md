# EECS 348 Lab 6

Web Programming: HTML, CSS, JavaScript, PHP

## Setup (cycle server)

1. Log into cycle server
2. Create `public_html` if it doesn't exist
3. Run:
   ```
   wget people.eecs.ku.edu/~l367r860/index.html
   wget people.eecs.ku.edu/~l367r860/grant_permission.sh
   bash grant_permission.sh
   ```
4. Replace the downloaded index with your own, or edit the wget index to link practice4.php instead of practice4.html (multiplication table needs PHP)
5. Upload all files to public_html
6. Run `bash grant_permission.sh` after adding new files

## Contents

- **practice1.html** - EECS profile (name, photo, contact)
- **practice2.html** + **practice2.js** - RGB color and border width control
- **practice3.html** + **practice3.js** - Password verification with alerts
- **practice4.php** - Multiplication table with row/column indexes

## Practice 1

Add your `profile.jpg` to the folder. Edit practice1.html with your name, email, and details. Replace the video embed URL if you want a different one.
